English Summary:

This file contains a set of beginner JavaScript examples organized into class sections. It demonstrates:
- Basic output with `console.log` and simple arithmetic (including a tax/KDV example).
- Primitive data types: numbers, strings, booleans, `undefined`, and `null`.
- Variable declaration and naming conventions (examples using camelCase and underscores).
- String handling: concatenation, template literals, and many string methods (`toLowerCase`, `toUpperCase`, `length`, indexing, `slice`, `substring`, `replace`, `trim`, `indexOf`, `split`).
- Example exercises: computing student math averages, checking pass/fail status, and parsing integers with `parseInt`.
- URL construction from a course name by lowercasing and replacing spaces with hyphens, and boolean checks like `startsWith` and `includes`.

The code is commented per class (Class 01..Class 06) and includes console outputs to show results of each operation.
